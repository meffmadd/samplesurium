Observable Plot

Interactive plots can be created using Observable Plot, which provides a powerful grammar of graphics for JavaScript.

data = {
  const co2data = await FileAttachment("data.csv").csv({ typed: true } );
  return co2data.map(d => { 
    d["decimal date"] = Number(d["decimal date"]);
    d.average = Number(d.average); 
    return d; 
  });
}
console.log(data);
Plot.plot({
  marks: [
    Plot.line(data, 
      { x: "decimal date", y: "average"}, 
      { stroke: "black" }
    )
  ]
})

The plot above shows the long-term trend of atmospheric CO₂ concentrations from 1958 to 2025, demonstrating the steady increase over time.
