A popular example for the use of regression (and which you will encounter in almost any intro stats course) is determining whether there is a correlation of height and weight in humans. We might try to fit a straight line to the data, which tells us how much the weight increases if a person is taller.

Now one thing to note is that there might be no correlation between the attributes that are analysed. In that case, the analysis will be meaningless for anything practical. In order to avoid that, we can use the correlation coefficient, which gives a measure of how much correlation exists between two variables.

In this example uses polynomial regression, so we are not limited to a line but a polynomial (here degree 4). The neural network tries to find the coefficients a, b, c and d of the function which fit the points best.
